Typora 如何修改字体和主题？

发表于 2021-09-12 分类于 Typora 阅读次数：

修改 Typora 的字体

有可能你不喜欢当前 Typora 的字体，想要更换，也有可能是因为你正在使用 Linux 系统，Typora 的字体默认是宋体，显得很难看。我们知道，Typora 默认的主题是 GitHub，那我们如果想要更改字体，只需要修改对应的 CSS 文件就可以了。首先点击 Typora 左上角的“文件”，在出现的菜单中选择“偏好设置”，然后导航到“外观”部分，就可以看到 Typora 当前正在使用的主题是什么了。点击“打开主题文件夹”，这是我们一般会看到如下画面：

如果我们正在使用 GitHub 主题，我们就只需要修改 github.css 这个文件就可以了，选中你正在使用的主题的配置文件即可。打开 github.css 之后，在里面找到类似于下面的配置：

然后修改font-family中的前两项就可以了，第一项可以用来显示西文字体，如：英文，法文，西班牙文；第二项可以用来显示中文字体、日文字体等，我这里改成了Fira Code和思源黑体。

修改 Typora 的主题

可以到 Typora 官方的主题网站下载主题，下载完后一般是一个压缩包，解压之后里面一般是一个文件夹和一个与文件夹同名的 CSS 文件，也有可能是单独的一个 CSS 文件。

安装方法

在 Typora 偏好设置的外观选项卡中，点击“打开主题文件夹”，然后将下载解压后的主题文件（和文件夹）拖入到其中，然后重启一下 Typora 即可，我这里使用了一个 Fluent 的主题。
